## Requirements

| Requirement        | Details                             |
| ------------------ | ----------------------------------- |
| Shader Loader      | OptiFine                            |
| GLSL Version       | 120                                 |
| Shader Type        | Lightweight color adjustment shader |
| Performance Impact | Minimal                             |

## Compatibility Notes

* Warmer White is designed to be lightweight and compatible with a wide range of Minecraft versions.
* The shader uses GLSL 120 to maximize compatibility with older hardware and Minecraft versions.
* Versions not listed above may work, but have not been officially tested.
* Other shader loaders may not provide full compatibility.

## Tested Features

* ✅ Shader loading
* ✅ Color temperature adjustment
* ✅ Normal Minecraft lighting behavior
* ✅ Stable rendering
* ✅ No noticeable performance impact
