# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added

* Initial release of **Warmer White** shader pack.
* Lightweight color temperature adjustment shader.
* Slightly warmer tones while preserving the original Minecraft lighting and visuals.
* Minimal performance impact.

### Compatibility Tested

Successfully tested on:

* Minecraft 1.14.4
* Minecraft 1.16.4
* Minecraft 1.21
* Minecraft 1.21.1
* Minecraft 1.21.3
* Minecraft 1.21.4
* Minecraft 1.21.6
* Minecraft 1.21.7
* Minecraft 1.21.8
* Minecraft 1.21.9
* Minecraft 1.21.11

### Notes

* Requires OptiFine with shader support.
* Uses GLSL 120 for maximum compatibility.
* Older and newer versions may work as long as they support the required shader features.
